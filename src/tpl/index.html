UI-kit RT.ru

Здесь собраны финальные классы и необходимая вёрстка для CMS.
Селекторная нагрузка стремится к БЭМ (.blok – блок, .blok__element – элемент, .blok_mod – модификатор.), но есть и хелперы для работы в CMS (.text-center, .color-red).
Формат классов вписан для emmet'а, скопипастив его и нажав таб сформирует необходимую строку (.block.block_mod -> class="block block_mod").
Стили на текущий момент обновляются и добавляются в бандл в сжатом виде, посредством LESS-препроцессора. Поэтому нет смысла их править через инлайн вставки.
Если стили ограничены или не хватает элементов, я с удовольствием их модифицирую.

.row

.row

На данный момент сетка включает 4 брейкпоинта (разрешения для адаптивности), названных как в привычном бутстрапе

[ sm ] - ширина экрана >= 576px

[ md ] - ширина экрана >= 768px

[ lg ] - ширина экрана >= 993px

[ xl ] - ширина экрана >= 1083px

[ xxl] - ширина экрана >= 1236px

Брейкпоинты используются для колонок, чтобы при определенном разрешении они перестраивались
